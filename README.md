# Flight Booking System (Java OOP Project) 

##  Description  
A console-based Flight Booking System developed using Java. It demonstrates Object-Oriented Programming concepts like Abstraction, Inheritance, and Encapsulation.  

---

##  Features
- Passenger registration
- Flight booking system
- Seat availability check
- Multiple payment methods (Card, UPI, Cash)
- Airline and flight details display

---

##  Tech Stack
- Java
- Object-Oriented Programming
- Scanner (Console Input)

---

##  How to Run
1. Compile the file:  
   javac FlightBookingSystem.java  
2. Run the program:  
   java FlightBookingSystem  
   
---

##  Concepts Used
- Classes & Objects
- Encapsulation
- Inheritance
- Abstraction
- Method Overriding
